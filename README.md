# Lab Data Formatter v1.4.2

HIS OPD 協助熱鍵 / 整理報告

## 簡介

Lab Data Formatter 是一個常駐系統匣的小工具，能自動擷取醫院檢驗報告文字，轉換成簡潔格式，方便貼到病歷或筆記中。

## 功能

- **Alt+1** — 全選 + 複製（任何視窗皆可使用）
- **Alt+2** — 貼上整理後的檢驗報告
- **Ctrl+0** — 開啟設定視窗
- **Ctrl+1~4** — 自訂快速貼上欄位
- 支援多份報告自動合併（15 秒內）
- 超過 80 字元自動換行
- 系統匣 LED 狀態指示

## 安裝

1. 下載 `LabFormatter.exe`
2. 直接執行，程式會常駐在系統匣
3. 無需安裝，設定檔自動建立於 `我的文件\EMR_Report\`
