# Lab Data Formatter v1.6.0

HIS OPD 協助熱鍵 / 整理報告

## 簡介

Lab Data Formatter 是一個常駐系統匣的小工具，能自動擷取醫院檢驗報告文字，轉換成簡潔格式，方便貼到病歷或筆記中。

## 功能

### 6 組可自訂熱鍵
- **Ctrl+1~4** — 自訂欄位（快貼 / 報告整理 / 範本 / 擷取）
- **熱鍵5（預設 Ctrl+E）** — 全選 + 複製
- **熱鍵6（預設 Ctrl+R）** — 貼上整理後的檢驗報告
- **Ctrl+0** — 開啟設定視窗

### 報告處理
- 支援多份報告自動合併（15 秒內）
- 超過 80 字元自動換行
- 系統匣 LED 狀態指示 + 浮動預覽

### 注意事項
- 建議使用 **Ctrl** 系列熱鍵，Alt 系列可能不穩定

## 安裝

1. 下載 `LabFormatter.exe`
2. 直接執行，程式會常駐在系統匣
3. 無需安裝，設定檔自動建立於 `我的文件\EMR_REPORT\`
