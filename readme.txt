【簡介】

  HIS OPD Data Assistant 是一個常駐系統匣的小工具，
  能自動整理醫院 ORD 系統及健保雲端的檢驗報告與藥歷資料，
  轉換成簡潔格式，方便貼到病歷或筆記中。

  免安裝、不連網、所有處理皆在本機記憶體完成。

================================================================

【v2.3 功能一覽】

  1. 滑鼠右鍵一鍵擷取（推薦）
     在報告視窗上按右鍵，程式自動讀取視窗資料並整理，
     結果直接寫入剪貼簿，Ctrl+V 即可貼上。
     全程無畫面閃動（WM_GETTEXT 直接從記憶體讀取）。

  2. 多份報告合併（右鍵和 Ctrl+C 均支援）
     15 秒內多次右鍵（或 Ctrl+C），報告自動累積合併，
     圖示顯示「✓ [2筆]」確認筆數。

  3. 右鍵快速開關熱鍵
     按 Ctrl+` 瞬間開啟/關閉右鍵擷取功能，
     右下角彈出狀態通知，不需進設定畫面。

  4. 右鍵限定視窗
     可在設定中填入視窗標題關鍵字（如 ORD），
     讓右鍵只在該視窗攔截，其他程式右鍵選單完全正常。

  5. 雙軌操作
     - 方法一：滑鼠右鍵 → Ctrl+V 貼上（推薦）
     - 方法二：Ctrl+A → Ctrl+C → Ctrl+3 貼上（傳統）

  6. 健保雲端檢驗報告解析
     11502(其他)AC:86(5.3),BUN/Cr:21/1.02,Na/K:140/4.2,ALT:18

  7. 健保雲端藥歷整理
     Amlodipine(5)QD, Pitavastatin(2)QD, Aspirin(100)QD

  8. ORD 慢箋 / 急性處方提煉
     複製整份處方單（含 SOAP 筆記也沒關係），程式自動偵測
     並提煉藥名+劑量+用法，輸出格式：
     Glimepiride (2) 1# QD, L-Thyroxine (100) 1# QD, Pitavastatin (2) 1# QD

  9. TSH / Free T4 甲狀腺功能報告
     ORD 或含 TSH/Free T4 的報告，輸出格式：
     TSH/FT4:0.055/1.11（正確處理 L/H 異常旗標）

  10. Ctrl+3 / 貼上鍵 精確對焦
      按下熱鍵時立即記錄目標視窗，用 SetForegroundWindow
      恢復焦點後再貼上，不受游標移動影響。

  11. 大量檢驗項目擴充（v2.3.15 新增）
      新增 30+ 個項目可在設定中啟用：
      - 凝血：PT、INR、APTT、D-Dimer
      - 心臟：Troponin、CK、CK-MB、BNP/NT-proBNP
      - 胰臟：Amylase、Lipase
      - 肝膽：ALP、GGT、D-Bil
      - 內分泌：T3、Cortisol
      - 維生素：Vitamin D、B12、Folate
      - 腫瘤：CEA、AFP、PSA、CA-199、CA-125、CA-153
      - 風濕：RF、ANA、Anti-CCP
      - CRP 從預設 OFF 改為預設 ON

  12. 啟動時檢查更新（v2.3.15 新增）
      偵測到 GitHub 上有新版本時，右下角彈出通知。
      可在設定中關閉。

  13. 設定匯出 / 匯入（v2.3.15 新增）
      換電腦或分享設定給同事用：
      設定畫面 → 匯出設定 → 將 JSON 檔複製到新電腦 → 匯入。

  14. 歷史記錄（v2.3.15 新增）
      最近 10 筆轉換結果保留在記憶體（程式關閉即清除），
      可在設定畫面查看、複製、清除。

  15. 病人 ID 自動遮罩（v2.3.15 新增）
      勾選後，輸出時自動將：
      - 健保 ID / 身分證（A123456789 → A*****56789）
      - 8+ 位連續數字（病歷號）
      做部分遮罩。預設關閉。

  16. 本機操作日誌（v2.3.15 新增）
      可選擇性記錄操作時間與動作類型（不含病人資料），
      存於 我的文件\EMR_Report\labformatter.log。

【安裝方式】

  1. 從網頁下載 LabFormatter_v2.3.15.exe
     下載網址：https://andrew22tw.github.io/HIS_OPD_DATA_Assistant/
  2. 放在桌面或任何位置
  3. 雙擊啟動，圖示出現在右下角系統匣

  免安裝，下載即用。

================================================================

【操作方式】

  ⭐ 方法一：右鍵擷取（推薦）

    步驟一：將滑鼠移到報告視窗上
    步驟二：按滑鼠右鍵 → 程式自動讀取並整理
    步驟三：圖示變綠，右下角彈出預覽視窗
    步驟四：在要貼上的地方按 Ctrl+V
           （若 Ctrl+V 失效，可用 Ctrl+3 / 貼上鍵代替）
    步驟五：如需合併多份報告，15 秒內在其他視窗再次右鍵

  方法二：複製擷取（傳統）

    步驟一：將滑鼠移到報告視窗上
    步驟二：按 Ctrl+A 全選，再按 Ctrl+C 複製
           （或使用擷取鍵一鍵完成）
    步驟三：圖示變綠，右下角彈出預覽視窗
    步驟四：15 秒內可繼續切換報告重複擷取（自動合併）
    步驟五：滑鼠移到目標視窗，按 Ctrl+3 貼上結果

================================================================

【操作方式 — 健保雲端】

  步驟一：開啟健保雲端，進入「檢查檢驗結果」或「西醫用藥」
  步驟二：Ctrl+A 全選頁面，Ctrl+C 複製
  步驟三：圖示變綠，右下角彈出預覽視窗
  步驟四：滑鼠移到目標視窗，按 Ctrl+3 貼上結果

================================================================

【熱鍵說明】

  滑鼠右鍵         一鍵擷取（讀取+整理+寫入剪貼簿）
  Ctrl+V           貼上右鍵擷取的結果
  Ctrl+`           切換右鍵擷取 開/關（右下角顯示狀態）
                   ※ 可在設定中修改此熱鍵
  Ctrl+A → Ctrl+C  全選+複製報告（自動辨識整理）
  擷取鍵（可自訂）  按下後自動全選+複製，預設 Ctrl+E
  Ctrl+3 / 貼上鍵  貼上整理結果（自動對焦後貼上）
                   Ctrl+V 失效時的備用方案
  Ctrl+0           開啟設定視窗
  Ctrl+1 ~ Ctrl+4  自訂快貼槽（可設文字片段或觸發報告貼上）

  ※ 擷取鍵、貼上鍵、右鍵切換鍵均可在設定視窗中自訂

================================================================

【設定視窗（Ctrl+0）說明】

  熱鍵設定
  - 擷取鍵：預設 Ctrl+E（一鍵觸發全選+複製）
  - 貼上鍵：預設 Ctrl+3

  滑鼠右鍵擷取
  - 啟用/停用 勾選框：即時生效
  - 限定視窗：填入視窗標題關鍵字（如 ORD），
              只攔截該視窗右鍵，其他程式右鍵選單正常
  - 切換熱鍵：預設 Ctrl+`，可改為其他組合

  輸出項目
  - 勾選要顯示的檢驗項目（AC、BUN、Cr、Na、K…等）

================================================================

【支援的資料來源】

  1. ORD 檢驗報告 → 濃縮格式（如 AC:122(6.5),BUN/Cr:27/1.71）
  2. 健保雲端檢驗 → 同上濃縮格式（加上月份及來源標記）
  3. 健保雲端藥歷 → 學名橫式格式（自動取學名、劑量、用法）

  程式會自動辨識資料類型，不需手動切換。
  隱私安全：不連網，病人資料不會外洩。

================================================================

【多份報告合併】

  同一位病人有多份報告（生化、血液、尿液）時：
  - 右鍵：15 秒內在各視窗右鍵，報告自動合併
  - Ctrl+C：15 秒內連續複製，報告自動合併

  程式自動區分血液和尿液檢體：
  - 血液項目（BUN、Cr、Na、K 等）只從「檢體 血液」擷取
  - 尿液項目（ACR、PCR）只從「檢體 尿液」擷取

================================================================

【輸出格式說明】

  ORD 檢驗輸出順序：
  AC → HbA1c → BUN/Cr → Na/K → HCO3 → ALT/AST → Alb →
  TC/TG/L/H → UA → Ca/IP → Iron/TIBC/Ferritin → Hb → ACR → PCR

  特殊規則：
  - AC 有 HbA1c 時合併顯示：AC:122(6.5)
  - BUN 自動取整數
  - 超過 80 字元自動換行

  藥歷輸出格式：
  - 自動提取學名（有學名優先顯示，無則顯示商品名）
  - 劑量括號：Amlodipine(5)
  - 超過 1 顆標示：Amlodipine(5) 2# QD
  - 過濾急診 STAT 注射，只保留一般處方藥物

================================================================

【系統匣圖示顏色】

  黃色    等待中（尚未擷取報告）
  藍色    右鍵擷取中（正在讀取視窗資料）
  綠色    已轉換（有結果可以貼上）
  紅色    錯誤（例如沒先擷取就按貼上鍵）

【常見問題】

  Q: 右鍵按了沒反應？
  A: 1. 確認設定中「滑鼠右鍵擷取」已勾選
     2. 若設定了「限定視窗」，確認目標視窗標題含有該關鍵字

  Q: 右鍵擷取和其他程式的右鍵選單衝突？
  A: 兩種解法：
     ① 按 Ctrl+` 快速切換開/關（推薦）
     ② 在設定的「限定視窗」填入 ORD 視窗標題關鍵字

  Q: Ctrl+V 貼不出來？
  A: 改用 Ctrl+3（貼上鍵），會自動點擊對焦後再貼上。

  Q: Ctrl+3 按了沒反應？
  A: 請先擷取報告（右鍵或 Ctrl+C），等圖示變綠再按。

  Q: 複製了但沒轉換？
  A: 確認複製的是檢驗報告文字，程式需要偵測到
     關鍵字（BUN、mg/dl、Cr、藥品名稱等）才會觸發。

  Q: 合併的結果不對？
  A: 確認在 15 秒內完成所有複製/右鍵動作。

  Q: 熱鍵和其他軟體衝突？
  A: 在設定視窗修改擷取鍵或貼上鍵。

  Q: 怎麼完全關閉程式？
  A: 右鍵系統匣圖示 → 結束
