  Lab Data Formatter v1.3.1 使用說明
  作者：吳岳霖醫師  DAL93@tpech.gov.tw

【簡介】

  Lab Data Formatter 是一個常駐系統匣的小工具，
  能自動擷取醫院檢驗報告文字，轉換成簡潔格式，
  方便貼到病歷或筆記中。

  例如把整頁報告轉成：
  11503 AC:122(6.5),BUN/Cr:27/1.71,Na/K:140/3.6,ALT:25

================================================================

【安裝方式】

  1. 從網頁下載 LabFormatter.exe 和 readme.txt
  2. 放在任何位置（桌面即可）
  3. 雙擊 LabFormatter.exe 啟動，圖示出現在右下角系統匣

  免安裝，下載即用。

================================================================

【基本操作】

  步驟一：在醫院系統中，開啟檢驗報告視窗
  步驟二：按 Ctrl+`（擷取鍵）自動全選+複製
  步驟三：系統匣燈泡變綠，右下角彈出預覽視窗
  步驟四：切到要貼上的地方，按 Ctrl+3 自動貼上

================================================================

【熱鍵說明】

  Ctrl+`    擷取（自動全選+複製，可在設定中自訂）
  Ctrl+0    開啟設定視窗
  Ctrl+1    自訂快貼 1（可設定常用文字片段）
  Ctrl+2    自訂快貼 2
  Ctrl+3    貼上檢驗報告轉換結果（預設）
  Ctrl+4    自訂快貼 3

  擷取鍵可在設定視窗中改成 Ctrl 或 Alt + 任意鍵。

================================================================

【多份報告合併】

  如果同一位病人有多份報告（例如生化、血液、尿液），
  只要在 15 秒內連續擷取 複製，程式會自動合併。

  例如：
    第 1 次 Ctrl+C → 複製生化報告 → 預覽顯示結果
    第 2 次 Ctrl+C → 複製 HbA1c 報告 → 自動合併，顯示 [2筆]
    第 3 次 Ctrl+C → 複製尿液報告 → 自動合併，顯示 [3筆]
    按 Ctrl+3 → 一次貼上全部結果

  超過 15 秒沒有新報告，下次複製視為新的一筆。

  程式會自動區分血液和尿液檢體：
  - 血液項目（BUN、Cr、Na、K 等）只從「檢體 血液」擷取
  - 尿液項目（UAC、UPC）只從「檢體 尿液」擷取
  - 不會互相混淆

================================================================

【輸出格式說明】

  日期格式：民國年YYYMM（例如 11503 = 民國 115 年 3 月）

  輸出順序：
  AC → HbA1c → BUN/Cr → Na/K → HCO3 → ALT/AST → Alb →
  TC/TG/L/H → UA → Ca/IP → Iron/TIBC/Ferritin → Hb → ACR → PCR

  自動分組（用 / 連接）：
  - BUN/Cr        腎功能
  - Na/K          電解質
  - ALT/AST       肝功能
  - TC/TG/LD/HD   血脂
  - Ca/IP         鈣磷

  特殊規則：
  - AC 有 HbA1c 時合併顯示：AC:122(6.5)
  - AC 沒有 HbA1c 時單獨顯示：AC:122
  - BUN 自動取整數
  - 只有 HbA1c 沒有 AC 時：HbA1c:6.5

  顯示名稱對照：
  - Glu AC       → AC
  - Cholesterol  → TC
  - LDL-C        → LD
  - HDL-C        → HD
  - Phosphate/P  → IP
  - Uric Acid    → UA
  - ACR/UACR     → UAC
  - UPCR/PCR     → UPC

================================================================

【設定視窗】

  按 Ctrl+0 或右鍵點系統匣圖示 → 設定

  每個熱鍵可設定為以下類型：
  - 快貼文字：按熱鍵自動貼上預設的文字
  - 報告整理：Ctrl+C 複製報告後，按此鍵貼上轉換結果
  - 範本套用：文字中的 {clipboard} 會被替換成剪貼簿內容
  - 未設定：此熱鍵不動作

  修改完按「儲存設定」。

  也可以直接用記事本編輯 JSON 設定檔，
  編輯完在設定視窗按「讀取」重新載入。

================================================================

【系統匣燈泡顏色】

  黃色    等待中（尚未複製報告）
  綠色    已轉換（有結果可以貼上）
  紅色    錯誤（例如沒有先複製報告就按 Ctrl+3）

================================================================

【檔案說明】

  LabFormatter.exe              主程式（下載即用）
  readme.txt                    本說明文件

  設定檔位置（自動建立）：
  我的文件\EMR_Report\lab_formatter_config.json

  每台電腦的設定檔獨立存放在個人文件資料夾，
  即使多人共用同一台電腦，各自的設定不會互相影響。

================================================================

【常見問題】

  Q: Ctrl+3 按了沒反應？
  A: 請先 Ctrl+C 複製報告，等燈泡變綠再按。

  Q: 複製了但沒轉換？
  A: 確認複製的是檢驗報告文字，程式需要至少 3 個
     關鍵字（如 BUN、mg/dl、Cr 等）才會觸發。

  Q: 合併的結果不對？
  A: 確認 10 秒內連續複製。如果超過 15 秒，
     前一筆會被清除，重新開始。

  Q: 熱鍵和其他軟體衝突？
  A: 可以在 JSON 設定檔中修改熱鍵配置。

  Q: 怎麼完全關閉程式？
  A: 右鍵系統匣圖示 → 結束
