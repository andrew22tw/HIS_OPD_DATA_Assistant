【簡介】

  Lab Data Formatter 是一個常駐系統匣的小工具，
  能自動擷取醫院檢驗報告文字，轉換成簡潔格式，
  方便貼到病歷或筆記中。

  例如把整頁報告轉成：
  1150322 AC:122(6.5),BUN/Cr:27/1.71,Na/K:140/3.6,ALT:25

================================================================

【安裝方式】

  1. 從網頁下載 LabFormatter.exe 和 readme.txt
     下載網址：https://andrew22tw.github.io/HIS_OPD_DATA_Assistant/
  2. 放在任何位置（桌面即可）
  3. 雙擊 LabFormatter.exe 啟動，圖示出現在右下角系統匣

  免安裝，下載即用。

================================================================

【熱鍵總覽 — 共 6 組可自訂熱鍵】

  本程式提供 6 組熱鍵，預設如下：

  熱鍵         預設         功能
  ─────────────────────────────────────────────
  Ctrl+1       快貼1        貼上自訂文字
  Ctrl+2       快貼2        貼上自訂文字
  Ctrl+3       濃縮報告     執行 Slot 3 設定的動作（預設：貼上整理後的檢驗報告）
  Ctrl+4       快貼3        貼上自訂文字
  Ctrl+E       擷取         一鍵全選+複製（取代 Ctrl+A → Ctrl+C）
  Ctrl+R       貼上         永遠貼上整理後的檢驗報告（不受 Slot 設定影響）

  另外 Ctrl+0 固定為開啟設定視窗。

  ※ Ctrl+3 是執行 Slot 3 所設定的功能（預設為報告整理），
    如果 Slot 3 被改成快貼或其他類型，按 Ctrl+3 就會執行該類型。
    Ctrl+R（熱鍵 6）則無論如何都會貼上檢驗報告整理結果。

  ※ 修飾鍵可在設定中選擇 Ctrl 或 Alt。
    建議使用 Ctrl，Alt 容易與其他程式衝突，不穩定。

================================================================

【什麼是「擷取」？】

  「擷取」= 自動全選 + 複製，一鍵完成兩個動作。

  原本你需要按兩次：Ctrl+A（全選）→ Ctrl+C（複製）
  設定擷取熱鍵後，只需按一次（預設 Ctrl+E）即可完成。

  擷取成功後，程式會自動辨識檢驗報告並整理，
  右下角會彈出綠色預覽視窗顯示結果。

================================================================

【基本操作 — ORD 系統】

  步驟一：將滑鼠移到 ORD 系統的檢驗報告上
  步驟二：滑鼠移到報告區，按 Ctrl+E 一鍵擷取
         （不需先點擊視窗，滑鼠移過去就能擷取）
         或手動 Ctrl+A 全選 → Ctrl+C 複製
  步驟三：系統匣醫生圖示變綠，右下角彈出預覽視窗
  步驟四：可繼續切換不同報告，重複步驟二持續擷取
         （15 秒內的報告會自動合併）
  步驟五：切到要貼上的地方，按 Ctrl+3 貼上整理結果

================================================================

【預覽視窗】

  擷取成功後，右下角會彈出綠色預覽視窗，顯示整理結果。

  - 單筆報告：8 秒後自動隱藏
  - 多筆合併報告：15 秒後自動隱藏
  - 可拖曳標題列移動視窗位置
  - 右上角有關閉按鈕（✕）可手動關閉

================================================================

【Slot 類型說明】

  Ctrl+1~4 每個熱鍵可設定為以下類型：

  類型         功能
  ─────────────────────────────────────────────
  快貼         按熱鍵自動貼上預設的文字
  報告         Ctrl+C 複製報告後，按此鍵貼上轉換結果
  擷取         按熱鍵自動全選+複製（同 Ctrl+E）
  未設定       此熱鍵不動作

================================================================

【多份報告合併】

  如果同一位病人有多份報告（例如生化、血液、尿液），
  只要在 15 秒內連續擷取，程式會自動合併。

  例如：
    Ctrl+E → 擷取生化報告 → 預覽顯示結果
    切換報告 → Ctrl+E → 擷取 HbA1c → 自動合併，顯示 [2筆]
    切換報告 → Ctrl+E → 擷取尿液報告 → 自動合併，顯示 [3筆]
    按 Ctrl+3 → 一次貼上全部結果

  超過 15 秒沒有新報告，下次擷取視為新的一筆。

  程式會自動區分血液和尿液檢體：
  - 血液項目（BUN、Cr、Na、K 等）只從「檢體 血液」擷取
  - 尿液項目（UAC、UPC）只從「檢體 尿液」擷取
  - 不會互相混淆

================================================================

【輸出格式說明】

  日期格式：民國年YYYMMDD（例如 1150322 = 民國 115 年 3 月 22 日）

  輸出順序：
  AC → HbA1c → BUN/Cr → eGFR → Na/K → HCO3 → CRP →
  ALT/AST → Alb → TC/TG/L/H → UA → Ca/IP →
  Iron/TIBC/Ferritin → WBC → PLT → TBil → Mg → Hb → ACR → PCR

  ※ eGFR、CRP 預設關閉，可在設定中勾選啟用。

  自動分組（用 / 連接）：
  - BUN/Cr        腎功能
  - Na/K          電解質
  - ALT/AST       肝功能
  - TC/TG/L/H     血脂
  - Ca/IP         鈣磷

  特殊規則：
  - AC 有 HbA1c 時合併顯示：AC:122(6.5)
  - AC 沒有 HbA1c 時單獨顯示：AC:122
  - BUN、HCO3、Iron、TIBC、Ferritin 自動取整數
  - 只有 HbA1c 沒有 AC 時：HbA1c:6.5

================================================================

【系統匣醫生圖示顏色】

  黃色    待機中（等待擷取報告）
  綠色    擷取成功（報告已整理，可貼上）
  紅色    錯誤（請先擷取報告）

================================================================

【設定視窗】

  開啟方式：
  - 按 Ctrl+0
  - 右鍵點系統匣圖示 → 設定
  - 雙擊系統匣圖示

  - 上方：Ctrl+1~4 的名稱、類型、內容
  - 中間：輸出項目勾選（取消勾選的項目不會出現在結果中）
  - 下方：熱鍵5（擷取）和熱鍵6（貼上）的按鍵設定

  修飾鍵可選 Ctrl 或 Alt。
  ⚠ 建議使用 Ctrl，Alt 與許多程式衝突，容易不穩定。

  修改完按「儲存設定」。

  也可以直接用記事本編輯 JSON 設定檔，
  編輯完在設定視窗按「讀取」重新載入。

================================================================

【右鍵選單】

  右鍵點系統匣醫生圖示，可以看到以下選項：

  - Ctrl+0 設定    開啟設定視窗
  - 編輯 JSON      用記事本開啟 JSON 設定檔直接編輯
  - 手動轉換       開啟手動轉換視窗
  - 使用說明       開啟本說明文件
  - 結束           關閉程式

================================================================

【手動轉換】

  右鍵點系統匣醫生圖示 → 手動轉換

  開啟手動轉換視窗後，可以將檢驗報告文字直接貼入輸入框，
  程式會即時轉換並顯示結果。

  適用於不方便使用熱鍵擷取的情況，
  例如從其他來源取得的報告文字。

================================================================

【檔案說明】

  LabFormatter.exe              主程式（下載即用）
  readme.txt                    本說明文件

  設定檔位置（自動建立）：
  我的文件\EMR_Report\lab_formatter_config.json

  每台電腦的設定檔獨立存放在個人文件資料夾，
  即使多人共用同一台電腦，各自的設定不會互相影響。

【常見問題】

  Q: Ctrl+3 按了沒反應？
  A: 請先擷取報告（Ctrl+E 或 Ctrl+C），等醫生圖示變綠再按。

  Q: 複製了但沒轉換？
  A: 確認複製的是檢驗報告文字，程式需要至少 3 個
     關鍵字（如 BUN、mg/dl、Cr 等）才會觸發。

  Q: 合併的結果不對？
  A: 確認 15 秒內連續擷取。超過 15 秒，
     前一筆會被清除，重新開始。

  Q: 熱鍵和其他軟體衝突？
  A: 在設定視窗修改熱鍵5/熱鍵6 的按鍵，
     或直接編輯 JSON 設定檔。

  Q: 怎麼完全關閉程式？
  A: 右鍵系統匣圖示 → 結束
