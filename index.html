6 組可自訂熱鍵

所有熱鍵皆可在設定視窗（Ctrl+0）中修改。修飾鍵可選 Ctrl 或 Alt。

⚠ 建議使用 Ctrl 作為修飾鍵。Alt 容易與其他程式（如 NVIDIA、瀏覽器）衝突，造成熱鍵不穩定。

什麼是「擷取」？

「擷取」= 自動點擊 + 全選 + 複製，一鍵完成三個動作。
原本需要點擊視窗 → Ctrl+A（全選）→ Ctrl+C（複製）三步驟，
v1.6.4 起只需將滑鼠移到目標視窗，按一次熱鍵（預設 Ctrl+E）即可全部完成。

Slot 類型

Ctrl+1~4 每個 slot 可在設定中切換為以下類型：

Slot 類型說明表
類型功能
快貼按熱鍵自動貼上預設的文字
報告貼上整理後的檢驗報告結果
擷取一鍵全選+複製（同 Ctrl+E 功能）
未設定此熱鍵不動作

系統匣狀態指示

程式啟動後會常駐在右下角系統匣，醫生圖示的十字會變色表示目前狀態：

待機中 — 等待擷取報告
擷取成功 — 報告已整理，可貼上
錯誤 — 請先擷取報告

多份報告自動合併

同一位病人有多份報告（生化、CBC、尿液）？只要在 15 秒內連續擷取，程式會自動合併成一行。

滑鼠移到生化報告 → Ctrl+E → 自動點擊+擷取 → 預覽顯示結果
滑鼠移到 HbA1c 報告 → Ctrl+E → 自動點擊+擷取 → 自動合併，顯示 [2筆]
滑鼠移到尿液報告 → Ctrl+E → 自動點擊+擷取 → 自動合併，顯示 [3筆]
按 Ctrl+3 → 一次貼上全部結果

血液項目（BUN、Cr、Na 等）只從血液檢體擷取；尿液項目（ACR、PCR）只從尿液檢體擷取，不會混淆。

安裝方式

1. 下載 LabFormatter.exe（約 60KB，免安裝）
2. 放在任何位置（桌面即可），雙擊執行
3. 醫生圖示出現在右下角系統匣，程式開始常駐
4. 設定檔自動建立於 我的文件\EMR_Report\
