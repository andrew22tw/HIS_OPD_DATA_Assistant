6 組可自訂熱鍵

所有熱鍵皆可在設定視窗（Ctrl+0）中修改。修飾鍵可選 Ctrl 或 Alt。

⚠ 建議使用 Ctrl 作為修飾鍵。Alt 容易與其他程式（如 NVIDIA、瀏覽器）衝突，造成熱鍵不穩定。
預設熱鍵預設功能說明
Ctrl+E擷取（全選+複製）一鍵完成 Ctrl+A → Ctrl+C，程式自動辨識並整理報告
Ctrl+R貼上結果將整理後的報告寫入剪貼簿並貼上
Ctrl+1快貼 1貼上自訂文字片段
Ctrl+2快貼 2貼上自訂文字片段
Ctrl+3濃縮報告貼上整理後的檢驗報告（預設報告 slot）
Ctrl+4快貼 3貼上自訂文字片段
Ctrl+0設定開啟設定視窗（固定，不可更改）

什麼是「擷取」？

「擷取」= 全選 + 複製，一鍵完成兩個動作。
原本需要 Ctrl+A（全選）→ Ctrl+C（複製）兩步驟，
設定擷取熱鍵後只需按一次（預設 Ctrl+E）即可完成。

Slot 類型

Ctrl+1~4 每個 slot 可在設定中切換為以下類型：

類型功能
快貼按熱鍵自動貼上預設的文字
報告貼上整理後的檢驗報告結果
擷取一鍵全選+複製（同 Ctrl+E 功能）
未設定此熱鍵不動作

系統匣狀態指示

程式啟動後會常駐在右下角系統匣，醫生圖示的十字會變色表示目前狀態：

待機中 — 等待擷取報告
擷取成功 — 報告已整理，可貼上
錯誤 — 請先擷取報告

多份報告自動合併

同一位病人有多份報告（生化、CBC、尿液）？只要在 15 秒內連續擷取，程式會自動合併成一行。

Ctrl+E → 擷取生化 → 預覽顯示結果
切換報告 → Ctrl+E → 擷取 HbA1c → 自動合併，顯示 [2筆]
切換報告 → Ctrl+E → 擷取尿液 → 自動合併，顯示 [3筆]
按 Ctrl+3 → 一次貼上全部結果

血液項目（BUN、Cr、Na 等）只從血液檢體擷取；尿液項目（ACR、PCR）只從尿液檢體擷取，不會混淆。

安裝方式

1. 下載 LabFormatter.exe（約 60KB，免安裝）
2. 放在任何位置（桌面即可），雙擊執行
3. 醫生圖示出現在右下角系統匣，程式開始常駐
4. 設定檔自動建立於 我的文件\EMR_Report\
